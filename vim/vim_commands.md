Show Line Number
:set numbers

Show File Explorer
:set Vexplorer

Buffers:
:ls     Show buffer
:bn     Next Buffer
:bp     Previous Buffer
:sb     Select Buffer
  :sb <TAB>   Shuffle through buffers to select with TAB
:bd     Delete Buffer
  :bd <TAB>   shuffle through buffers to close with TAB
  
Search & Replace
:%s/<SEARCHSTRING>/<REPLACESTRING>/g

g for whole document
